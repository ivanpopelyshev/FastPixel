(

chainable

"Drawn" each layer to main dataLayer layer (Back-to-front); Will use visible layers from "layerList" parameter; Or visible layers from internal "layerList" property; Note: if "layerList" is empty or if there are no visible layers the model would be reseted; Also, notify subscribers.

    [in]

    • start Point | Undefined
    • offset Point | Undefined

    [in] List of layers to merge.
